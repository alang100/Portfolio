Bird Species Classification

This project applied the use of machine learning techniques to classify 10 bird species using a dataset of images. Developed models including Random Forest, SVM, and CNN were evaluated using techniques like standardization and cross-validation.

The CNN model achieved the highest accuracy at 83% on the test set, outperforming Random Forest (64%) and SVM (69%). Future work will include dataset expansion and further advanced modelling techniques.

Modelling libraries include Scikit-learn and Tensorflow.
